# IPL Analytics Dashboard 🏏

**Tools:** Power BI, DAX  
**Data:** 2008–2025 IPL match data  

## Overview
Interactive Power BI dashboard analyzing 17 seasons of IPL data 
with KPI cards, venue analysis, team performance charts, and Top N player insights.

## Key Features
- Team-wise win/loss performance analysis
- Top run scorers and wicket takers (Top N filter)
- Venue-wise match and performance analysis
- Season-wise trend analysis using slicers
- Advanced DAX measures for dynamic KPIs
